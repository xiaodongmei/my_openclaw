# 小红书互动评论

## 功能

1. 对指定笔记进行点赞
2. 生成20条非重复的经典高质量评论
3. 自动回复评论进行互动引流

## 工作流程

### 1. 目标笔记获取

通过 web_search 或 web_fetch 获取目标笔记内容：

web_search query: site:xiaohongshu.com [关键词]
web_fetch url: 笔记链接

分析笔记的主题、风格、受众特征。

### 2. 点赞操作

使用 xiaohongshu-mcp 技能的 browser 自动化功能对笔记进行点赞。
需要先通过 xiaohongshu-mcp 登录小红书创作者平台。

### 3. 生成20条经典评论

根据笔记内容生成20条不重复的经典评论。评论分为以下类型：

#### 共鸣型（4条）
- 表达强烈认同感
- 例：太有同感了！我也是这样觉得的~
- 例：说到心坎里了，每次都是这样

#### 请教型（4条）
- 真诚提问引发互动
- 例：姐妹求问XX在哪里买的呀？
- 例：想知道这个用了多久有效果呢

#### 分享型（4条）
- 分享自己的相关经验
- 例：我之前也试过，真的超好用！
- 例：我用的是XX，效果也不错推荐试试

#### 夸赞型（4条）
- 真诚不谄媚的赞美
- 例：拍得好好看！构图太绝了
- 例：这个分享太实用了，马上收藏

#### 补充型（4条）
- 提供有价值的补充信息
- 例：补充一下，这个搭配XX效果更好哦
- 例：同类型的还有XX也很不错，可以对比看看

### 评论生成原则

- 每条评论 15-50 字，简洁有力
- 口语化自然表达，不像机器人
- 20条评论之间内容不重复、角度不同
- 适当使用emoji但不过度（每条0-2个）
- 避免过于营销化的措辞
- 包含引导对方回复的问句（至少5条含提问）
- 评论要与笔记内容高度相关
- 避免敏感词和违规内容

### 4. 评论发布

通过 browser 工具或 xiaohongshu-mcp 自动在笔记下方发布评论。
每条评论间隔30-60秒，模拟真人操作节奏。

## 注意事项

- 遵守小红书社区规范，不发布违规内容
- 评论间隔自然，避免被识别为机器操作
- 评论内容真诚有价值，不做无意义刷评
- 优先选择与自身账号定位相关的笔记互动
