# 小红书爆款采集与写作

## 工作流程

### 1. 爆款内容采集

使用 web_search 工具搜索小红书热门内容：

web_search query: site:xiaohongshu.com 爆款 [话题关键词], maxResults: 20

使用 web_fetch 抓取具体笔记内容进行分析：

web_fetch url: 目标笔记URL

### 2. 爆款模式分析

分析采集到的爆款笔记，提取以下要素：
- 标题公式：数字+痛点+解决方案、反问句式、情绪共鸣
- 封面风格：对比图、清单图、教程图、前后对比
- 内容结构：开头hook - 痛点共鸣 - 干货输出 - 行动号召
- 标签策略：热门话题标签 + 精准长尾标签
- 发布时间：工作日12:00-13:00、18:00-21:00；周末10:00-22:00

### 3. 爆款文章生成

#### 标题模板

1. 数字型：XX个XXX，第X个绝了！
2. 对比型：XXX vs XXX，差距太大了
3. 痛点型：XXX的人一定要看这篇！
4. 反问型：为什么XXX？原来是因为…
5. 情绪型：救命！这个XXX也太好用了吧
6. 教程型：手把手教你XXX，小白也能学会
7. 清单型：我的XX好物清单｜用了X年的真心推荐

#### 正文结构

emoji 开头hook（1-2句，制造好奇/共鸣）
emoji 痛点描述（你是不是也遇到过...）
emoji 核心内容（分点列出，每点简洁有力）
emoji 使用心得/效果展示
emoji 总结/行动号召（点赞收藏不迷路~）

话题标签 热门标签 精准标签 长尾标签

#### 写作要点

- 字数控制在 300-800 字
- 多用 emoji 分段，视觉感强
- 口语化表达，像跟朋友聊天
- 加入个人体验和真实感受
- 适当使用网络热词和流行语
- 结尾引导互动（评论区聊聊/你们觉得呢）

### 4. 标签生成

每篇笔记生成 5-10 个标签：
- 2-3 个热门大标签（如 好物推荐 生活分享）
- 2-3 个精准中标签（如 护肤心得 平价好物）
- 2-3 个长尾小标签（如 油皮亲妈面霜 学生党护肤）
